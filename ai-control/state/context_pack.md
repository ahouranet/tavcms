# Context Pack (خلاصه وضعیت پروژه)
- در این تسک قراردادهای مشترک هسته برای فاز بوت‌استرپ تکمیل شدند.
- سرویس اسلاگ در `apps/core/slug.py` ایجاد شد و قوانین فارسی/انگلیسی را به‌صورت واحد اعمال می‌کند.
- اسلاگ فارسی حروف فارسی را حفظ می‌کند، فاصله را به `-` تبدیل می‌کند و نویز را حذف می‌کند.
- اسلاگ انگلیسی lowercase می‌شود و از پاکسازی استاندارد استفاده می‌کند.
- یکتایی اسلاگ به‌صورت language-aware پیاده‌سازی شد و suffixهای `-2`, `-3` ... با truncate امن تولید می‌شوند.
- میکسین `SlugMixin` اضافه شد تا همه مدل‌های محتوایی آینده بدون re-implementation از قرارداد اسلاگ استفاده کنند.
- ابزار تاریخ `apps/core/dates.py` اضافه شد: `format_date` و `format_datetime`.
- نگاشت نمایش تاریخ زبان‌محور است: `fa` جلالی، `en` میلادی، `ar` پشتیبانی پایه.
- مدل Singleton تنظیمات سراسری (`GlobalSettings`) با فیلدهای زبان پیش‌فرض، طول اسلاگ، و نگاشت زبان→تقویم اضافه شد.
- ثبت ادمین برای `GlobalSettings` انجام شد و ایجاد رکورد دوم در UI مسدود است.
- migration اولیه اپ core برای `GlobalSettings` ساخته شد.
- تست‌های جدید برای auto slug، یکتایی suffix، truncate طول اسلاگ، و فرمت تاریخ اضافه و پاس شدند.
- فاز بعدی باید روی helperهای i18n تکمیلی یا ادامه backlog فاز صفر متمرکز بماند و Phase1 شروع نشود.
