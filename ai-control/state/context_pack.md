# Context Pack (خلاصه وضعیت پروژه)
- نسخه‌های هدف ثابت مانده‌اند: Python 3.12.x و Django>=5.2,<5.3.
- Phase0-03 انجام شد: پیکربندی پایگاه‌داده پروژه روی PostgreSQL اعمال شد.
- در `config/settings/base.py` اتصال DB با متغیرهای محیطی `POSTGRES_*` تعریف شده است.
- وابستگی `psycopg[binary]` برای اتصال PostgreSQL به requirements اضافه شد.
- فایل `.env.example` برای الگوی تنظیمات DB/secret اضافه شده است.
- راهنمای اجرای migration و ساخت superuser برای PostgreSQL در README به‌روزرسانی شد.
- مسیرهای i18n فاز قبلی دست‌نخورده باقی ماندند و تغییری در منطق Phase0-02 ایجاد نشد.
- این تسک فقط روی آماده‌سازی DB متمرکز بود و وارد Phase0-04 نشد.
- گام بعدی مطابق backlog: Phase0-04 (تست/تثبیت i18n URL prefix برای همه routeها).
- قراردادهای core.slug و core.dates هنوز شروع نشده‌اند و در Phase0-05 قرار دارند.
- برای محیط واقعی باید مقادیر `POSTGRES_*` و `DJANGO_SECRET_KEY` از secrets manager تامین شوند.
